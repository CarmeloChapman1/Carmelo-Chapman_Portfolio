Fantasy Baseball Discord Bot

A Discord bot designed to support fantasy baseball decisions by surfacing stats and information directly inside a server.

Project Overview

This project connects my interest in baseball with my interest in software development. The goal of the bot is to make it easier for fantasy players to quickly look up stats and information without leaving Discord.

The bot listens for commands, fetches player data, and returns clean, easy-to-read responses in the сервер. It helped me think about how people actually use tools in a real-time chat environment.
