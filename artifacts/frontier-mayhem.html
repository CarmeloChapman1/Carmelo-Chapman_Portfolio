Frontier Mayhem (Unity Game)

A side-scrolling action game prototype built in Unity that combines beat-’em-up mechanics with a darker, modern atmosphere.

Project Overview

Frontier Mayhem is a personal and course-inspired project where I experimented with character design, combat logic, and level progression in Unity. The goal was to create a game that felt responsive and challenging, while still being readable and fair to the player.

The project features multiple character archetypes, basic enemy AI, and early ideas for a progression system. It gave me a chance to connect my interest in games with real programming practice.
